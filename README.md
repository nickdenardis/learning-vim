# VIM Basics

## Movement

`h` = Left
`j` = Down
`k` = Left
`l` = Up

## Editing
`x` = Delete letter under cursor
`i` = Edit mode
`a` = Append mode
`dd` = Delete whole line
`ESC` = Back to command mode

## Saving
`:q!` = Quit without saving
`:w` = Write file
`:wq` = Write and quit

## Useful
`:! <command>` = Run a terminal command
